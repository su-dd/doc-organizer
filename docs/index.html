🚀 快速开始

1

安装依赖

git clone https://github.com/su-dd/doc-organizer.git
cd doc-organizer
pip install -r requirements.txt
2

安装 Ollama

# 访问 https://ollama.ai 下载
ollama pull qwen2.5:7b
3

开始整理

python main.py
# 或使用命令行
python cli.py classify ./docs --output results.json
